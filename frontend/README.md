## Setup
1. `npm ci`
2. Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` to your Render backend URL.
3. `npm run dev`

## Deploy (Vercel)
- Import this folder as a Vercel project, Framework: Vite
- Set env var `VITE_API_BASE_URL`
- Build command: `npm run build`
- Output: `dist`
